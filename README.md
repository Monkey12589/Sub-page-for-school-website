# School Website Sub-Page - Responsive UX/UI Redesign

A clean, responsive concept design for a school website's sub-page, built completely with **pure HTML and CSS**. This project showcases modern user interface (UI) principles and fluid layouts without relying on any JavaScript frameworks or scripts.

The live prototype is deployed and viewable at:  
👉 **[Live Demo on Netlify](https://sub-page-for-school-website.netlify.app/)**

## 📐 UX/UI Design Focus
The core objective of this design was to optimize the readability and structure of school-related information. Because school websites often handle complex data (schedules, announcements, or departmental info), the user interface was designed from the ground up to prioritize **visual hierarchy** and **stress-free navigation**.

### ✨ Layout & UI Highlights:
*   **100% Pure CSS Architecture:** All components, layout flows, and interactive hover states are managed completely through CSS—zero JavaScript footprint for maximum speed and simplicity.
*   **Responsive Web Design (RWD):** Uses fluid grids and CSS media queries to ensure the interface automatically adapts, looking great on desktops, tablets, and mobile screens alike.
*   **Scannable Content Hierarchy:** Tailored typography sizes and clean section margins allow students, parents, and teachers to scan and find crucial information in seconds.
*   **Modern Visual Refresh:** A clean aesthetic that replaces outdated school website patterns with a fresh, spacious design.

## 🛠️ Built With
*   **HTML5** – Strict semantic elements to ensure high web accessibility and clear page architecture.
*   **CSS3** – Advanced layout styling, custom typography settings, media queries for responsiveness, and pure CSS interaction states.
